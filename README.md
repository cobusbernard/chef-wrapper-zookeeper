# chef-wrapper-zookeeper-cookbook

Wrapper cookbook to install Zookeeper.

## Supported Platforms

Uubntu 14.04

## Usage

### chef-wrapper-zookeeper::default

Include `chef-wrapper-zookeeper` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[chef-wrapper-zookeeper::default]"
  ]
}
```
